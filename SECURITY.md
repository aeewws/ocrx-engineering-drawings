# Security

## Supported Use

This project is intended for local desktop use on your own machine.

## Privacy

- The tool does not upload files by itself.
- The repo does not include sample customer drawings.
- The installer only writes into your Codex home, Codex bin directory, and optional AGENTS hint file.

## Sensitive Files

Do not commit:

- customer drawings
- private OCR outputs
- credentials or tokens
- user-specific absolute paths

## Reporting

If you find a privacy or security issue, open a private report before publishing details.
